# Iniciar o projeto Django

```
python -m venv venv
. venv/bin/activate
pip install django
django-admin startproject projeto .
```

# Configurar o git

```
git config --global user.name "seu_nome"
git config --global user.email "seu_email"
git config --global init.defaultBranch main
git init
```
